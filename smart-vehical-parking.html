Smart Vehicle Parking with Automatic Number Plate Recognition

Basic Introduction

    The Vehicle Parking System with Automatic Number Plate Recognition (ANPR) is designed to address urban parking challenges like traffic congestion and lack of available spaces. It uses image processing, computer vision, and IoT technologies to automate parking space management.

    When a vehicle arrives, a camera captures the license plate image, and image processing algorithms (like Bilateral Filtering, Canny Edge Detection, and Contour Detection) extract and convert the plate number into a string. The system then checks if the vehicle is registered and assigns an available parking slot if conditions are met, sending the slot number via email notification.

    For the admin interface, a Java desktop application manages customer data and vehicle information, performing full CRUD operations. The admin can monitor parked vehicles, check parking slot availability, and analyze data on parking usage, busiest hours, and customer activity.

    The system uses a combination of Python (OpenCV) for image processing, Java (JSP, Servlets) for user management, and hardware like Arduino, DC motors , and sensors for gate control. It enhances efficiency, security, and user experience by reducing waiting times and unauthorized parking.

    Despite some challenges like camera quality and budget constraints, the team created a functional prototype with plans for future improvements, like cloud database integration and higher-quality cameras. This system is a scalable and innovative solutionfor modern urban infrastructure

The Smart Vehicle Parking System is a well-designed, innovative solution for urban parking issues. It smartly combines image processing, ANPR, Java, Python, and IoT for efficient slot management and security.
